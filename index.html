Analog Devices Hardware Python Interfaces¶

This module provides a convenient way to access and control ADI hardware from Python through existing IIO drivers.

Requirements

  • libiio
  • libad9361 for AD9361 specific devices

class adi.dds.dds¶

DDS Signal generators: Each reference design contains two DDSs per channel. this allows for two complex tones to be generated per complex channel.

dds_enabled¶

DDS generator enable state

dds_frequencies¶

Frequencies of DDSs in Hz

dds_scales¶

Scale of DDS signal generators
